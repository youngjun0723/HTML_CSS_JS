레이아웃 중앙 정렬

.card-container를 사용해 화면 전체 너비와 높이의 정중앙에 프로필 카드가 위치하도록 만드세요. (힌트: Flexbox 또는 Grid와 min-height: 100vh)

배경색은 연한 회색(#f0f2f5)으로 설정하세요.

카드 스타일링

.profile-card는 너비 350px, 배경색 흰색(#ffffff), 모서리 둥글게(16px) 설정하세요.

자연스러운 그림자 효과(box-shadow)를 넣어 카드가 떠 있는 듯한 느낌을 주되, 전체적으로 안쪽에 적당한 여백(padding: 24px)을 부여하세요.

카드 내부 요소들은 세로로 정렬되고 가운데 정렬이어야 합니다.

이미지 스타일링

.profile-img를 완벽한 원형으로 만들고, 크기는 120px * 120px로 지정하세요.

테두리에 두께 4px짜리 실선(solid)을 두르고 색상은 주황/보라 계열의 포인트 컬러를 사용해 보세요.

텍스트 타이포그래피

이름(.user-name)은 글자 크기 24px, 두껍게(bold), 색상은 어두운 회색(#333333)으로 설정하세요.

직무(.user-role)는 글자 크기 14px, 색상은 파란색 또는 보라색 포인트 컬러를 쓰고 약간의 아래 여백을 주세요.

본문(.user-bio)은 글자 크기 15px, 색상 회색(#666666), 자간과 행간(line-height: 1.6)을 조절해 가독성을 높이세요.

버튼 그룹 스타일링

.card-footer 내부의 버튼들을 가로 방향으로 정렬하고, 버튼 사이에 적절한 간격(gap)을 주되 카드 너비에 맞춰 균등하게 배치해 보세요.

버튼 테두리를 없애고, 모서리를 둥글게(8px), 패딩을 주어 누르기 좋은 크기로 만드세요.

.btn-primary는 배경색을 포인트 컬러로, 글자색은 흰색으로 만드세요.

.btn-secondary는 배경색을 연한 회색으로, 글자색은 어두운 회색으로 만드세요.

인터랙션 (Hover 효과)

버튼 두 개에 마우스를 올렸을 때(:hover), 배경색이 살짝 어두워지거나 투명도가 변하는 효과를 부드럽게(transition) 적용하세요.
